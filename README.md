# Nomisma

Personal finance tracker — accounts, transactions, budgets, and recurring reminders.

## Stack

- **TanStack Start** — full-stack React framework with SSR
- **Convex** — real-time backend and database
- **Better Auth** — authentication with Google OAuth
- **shadcn/ui** — UI components and utilities
- **Tailwind CSS v4** — styling

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run typecheck
npm run check
npm run convex:dev
npm run convex:codegen
```

`npm run check` is the baseline verification command.

## Environment

Required environment variables:

- `CONVEX_DEPLOYMENT`
- `VITE_CONVEX_URL`
- `VITE_CONVEX_SITE_URL`
- `SITE_URL`
- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`

## Generated Files

- `convex/_generated/*` — generated by Convex
- `src/routeTree.gen.ts` — generated by TanStack Start

Do not edit generated files by hand.

## Local Setup

1. Install dependencies with `npm install`.
2. Copy `.env.example` to a local env file and fill in the required values.
3. Start Convex with `npm run convex:dev`.
4. Start the app with `npm run dev`.
